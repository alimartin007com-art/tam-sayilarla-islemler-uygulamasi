# 🎮 EMO v3.0 - Tam Sayılarla İşlemler

**"Tam Sayılarla İşlemler" (Eğlenceli Matematik Oyunu)**, öğrencilerin ve matematik tutkunlarının temel işlemleri eğlenerek pekiştirmesi için tasarlanmış, tarayıcı tabanlı interaktif bir eğitim uygulamasıdır.

👉 **[Hemen Oyna (Canlı Demo)](https://alimartin007com-art.github.io/tam-sayilarla-islemler-uygulamasi/)**

## ✨ Özellikler

Oyun sadece bir soru-cevap uygulaması değil, tam teşekküllü bir deneyim sunar:

- **🔥 Çeşitli Oyun Modları:**
  - **Klasik İşlemler:** Toplama, çıkarma, çarpma, bölme, üslü sayılar ve cebir.
  - **⚡ Blitz Modu (60sn):** 60 saniye içinde yapabildiğin kadar doğru yap!
  - **⏳ Hayatta Kalma:** Süreye karşı yarış; her doğru cevap ekstra zaman kazandırır.
  - **❓ Eksik Sayı Bulma:** Denklemlerdeki 'X' değerini bularak cebirsel zekanı test et.
- **☁️ Bulut Kaydı & Liderlik Tablosu:** Firebase entegrasyonu sayesinde hesabına giriş yap, ilerlemeni buluta kaydet ve Küresel Liderlik Tablosunda yerini al.
- **🎖️ XP ve Seviye Sistemi:** Soru çözdükçe XP kazanır ve seviye atlarsın. Seviye atladıkça ödüller kazanırsın.
- **🛒 Mağaza ve Ekonomi:** Doğru cevaplarla kazandığın 🪙 Bakiye ile mağazadan güçlendirmeler (2x XP, Kalkan vb.) ve yeni temalar satın alabilirsin.
- **🎨 Özelleştirilebilir Temalar:** Matrix, Cyberpunk, Sakura, Retro gibi farklı görsel temalarla oyunu kendi zevkine göre kişiselleştir.
- **🏅 Rozetler (Başarımlar):** Belirli görevleri tamamlayarak koleksiyonuna havalı rozetler ekle.
- **🤖 Adaptif Zorluk:** Oyun senin performansına göre soruların zorluğunu otomatik olarak ayarlar.

## 🛠️ Kullanılan Teknolojiler

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Backend & Veritabanı:** Firebase (Authentication, Realtime Database)
- **Ekstralar:** Canvas Confetti (Görsel efektler)

## 🚀 Kurulum & Yerel Çalıştırma

Eğer projeyi kendi bilgisayarında geliştirmek istersen:

1. Bu depoyu bilgisayarına klonla:
   ```bash
   git clone https://github.com/alimartin007com-art/tam-sayilarla-islemler-uygulamasi.git
   ```
2. Klasöre gir:
   ```bash
   cd tam-sayilarla-islemler-uygulamasi
   ```
3. Projeyi bir yerel sunucu (Local Server) ile çalıştır. (Firebase yetkilendirme servislerinin düzgün çalışması için `file://` protokolü yerine `http://localhost` üzerinden çalışması gerekir).
   - VS Code kullanıyorsan **"Live Server"** eklentisiyle açabilirsin.
   - Veya terminal üzerinden:
     ```bash
     npx http-server
     ```
